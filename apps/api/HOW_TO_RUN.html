Setup Steps

1
Navigate to the project folder
Open a terminal and change into the backend directory.
bash 
cd "C:\Users\IN7934\Desktop\2026\March-2026\backend_fronted_connector_files"
2
Create a virtual environment
Isolates project dependencies from your global Python installation.
bash 
python -m venv venv
3
Install dependencies
Installs pandas, FastAPI, and Uvicorn into the virtual environment.
bash 
# Windows
venv\Scripts\pip install pandas fastapi uvicorn

# macOS / Linux
venv/bin/pip install pandas fastapi uvicorn
pandas fastapi uvicorn
4
Run the ETL pipeline
Processes the CSV and writes 12 pre-computed JSON files to output/. Re-run whenever source data changes.
bash 
# Windows
venv\Scripts\python.exe etl_pipeline.py

# macOS / Linux
venv/bin/python etl_pipeline.py
expected output
Loaded 1000 rows | date range: 2026-02-18 to 2026-03-18
  [01/12] Written: dashboard_1d_all.json
  [02/12] Written: dashboard_1d_chat.json
  ...
  [12/12] Written: dashboard_30d_email.json
All 12 dashboard JSON files saved to 'output/'
5
Start the API server
Starts Uvicorn with hot-reload. Leave this terminal open while your frontend is running.
bash 
# Windows
venv\Scripts\python.exe api.py

# macOS / Linux
venv/bin/python api.py
expected output
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     Application startup complete.
● Live at localhost:8000 Hot-reload enabled CORS open to *

API Endpoint

GET http://localhost:8000/api/dashboard
Parameter Default Valid Values Description
time_window "7d" 24h
7d
30d		 Time range for the dashboard data
channel "all" all
chat
call
email		 Support channel filter
Example Requests
Frontend fetch — JavaScript 
// Default — 7 days, all channels
fetch('http://localhost:8000/api/dashboard')

// 30-day Chat data
fetch('http://localhost:8000/api/dashboard?time_window=30d&channel=chat')

// 24-hour All channels
fetch('http://localhost:8000/api/dashboard?time_window=24h&channel=all')
📖  Interactive Swagger docs available at http://localhost:8000/docs
