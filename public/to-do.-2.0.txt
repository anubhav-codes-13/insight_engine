# Prompt for Antigravity

Build a **modern enterprise-grade web application** using **Next.js 16 (App Router)** for an **AI-Powered Enterprise Customer Support Platform**.

The goal is to create a **visually impressive, scalable demo application** that showcases how AI can replace expensive human customer support operations.

This application should include **two main interfaces**:

1. Customer Interface
2. Admin Analytics Dashboard

The system should simulate how AI agents resolve customer issues automatically and provide analytics insights to enterprises.

The UI must be **clean, modern, enterprise-grade, and easy to use for non-technical users**.

Use **TypeScript, TailwindCSS, and modern component architecture**.

# Overall Product Concept

This platform demonstrates how enterprises can reduce customer support costs by using **AI agents instead of human support representatives**.

Typical enterprises receive **20,000–30,000 support calls per month**.

Each call costs approximately **$18**.

Monthly support costs can reach **$360,000 – $540,000**.

The AI system automatically resolves common issues such as:

Payment completed but order not created

Card payment failures

Refund status queries

Login issues

OTP failures

Delivery tracking problems

Checkout errors

Only complex cases should escalate to human support.
